# bootdev-blog-aggregator
Project that builds a newsfeed aggregator using Go and Postgres

Key Learnings
- Build out a CLI interface that users can interact with, defining and handling various commands that users input
- Configure a Postgres database of different users and their respective feeds
- Learn about Goose for migrations and sqlc for go code generation based on desired sql queries
- Build SQL Postgres Database with Multiple Tables and Foreign Key Connections
- Make HTTP Requests and parse the XML outputs
